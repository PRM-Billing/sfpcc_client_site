Companion: US-35-DASHBOARD-PFT-PST-STATUS-OVERVIEW-USER-STORY.md
User story

As a ClinicSync user on the home dashboard,
I want PFT and PST status overview sections like X-Ray and Sleep Study,
So that I can see order counts by stage at a glance and open the filtered orders list with one click.

Placement: New row below the existing X-Ray and Sleep Study status overview row.

Note: Activity Overview and Document Status charts on the dashboard stay unchanged (appointments / documents only). This story adds only the PFT and PST five-count panels.

Five metric cards (each module)

Card label (UI) What it counts When user clicks
Total Orders All active (non-archived) orders in the module Opens full PFT or PST orders list
Pending Orders waiting for report upload Filters list to pending_for_upload
In Progress Orders ready for review only (not “queued for CureMD”) Filters list to ready_for_review
Signed Signed orders not yet fully uploaded Filters list to signed
Uploaded to CureMD Orders completed in CureMD Filters list to uploaded_to_curemd

Counts come from existing APIs: GET /api/pft/orders/stats and GET /api/pst/orders/stats. Doctors acting on behalf of another physician see counts scoped to that physician (same as X-Ray / Sleep).

  • No empty or broken panel for users who lack module access — panel is hidden.
